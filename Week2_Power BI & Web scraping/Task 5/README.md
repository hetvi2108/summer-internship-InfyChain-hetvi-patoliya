# 🎬 Netflix Dashboard – Power BI

## 📌 Project Overview

This project presents an interactive **Power BI Dashboard** developed using the Netflix dataset to analyze content trends and generate meaningful insights. The dashboard enables users to explore Netflix's library through dynamic visualizations and filters, supporting data-driven decision-making.

---

## 🎯 Objectives

* Analyze the distribution of Netflix Movies and TV Shows.
* Identify trends in content releases over the years.
* Explore content ratings and genre popularity.
* Understand country-wise contributions to Netflix's catalog.
* Build an interactive dashboard for effective data storytelling.

---

## 🛠️ Tools & Technologies Used

* **Power BI Desktop**
* Power Query
* DAX (Data Analysis Expressions)
* Microsoft Excel / CSV Dataset

## 📊 Dashboard Features

* 📌 Total Movies and TV Shows KPI Cards
* 🌍 Country-wise Content Analysis
* 📅 Content Release Trends by Year
* ⭐ Content Rating Distribution
* 🎭 Genre Analysis
* 🔍 Interactive Filters and Slicers
* 📈 Dynamic Visualizations for Better Insights

---

## 🔍 Key Insights

* Movies constitute a larger portion of Netflix's content library than TV Shows.
* Netflix experienced significant growth in content additions after 2015.
* The United States contributes the highest number of titles.
* TV-MA and TV-14 are among the most common content ratings.
* Interactive filters allow users to explore insights from different perspectives.

## 🚀 How to Use

1. Download and open `Netflix_Dashboard.pbix` using **Power BI Desktop**.
