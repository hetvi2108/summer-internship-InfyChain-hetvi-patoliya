# 🎬 Netflix Data Analysis & Power BI Dashboard

## 📌 Project Overview
This project combines Exploratory Data Analysis (EDA) using Python and an interactive Power BI Dashboard to analyze Netflix content.

## 🎯 Objectives
- Data cleaning and preprocessing
- Analyze Netflix trends using Python
- Identify patterns in ratings, genres, and release years
- Build an interactive Power BI dashboard

## 🛠️ Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- Power BI
- DAX
