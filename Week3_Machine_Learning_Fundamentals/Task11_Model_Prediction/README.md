# 🎓 Student Performance Prediction

## 📌 Project Overview

This Machine Learning project predicts whether a student will **Pass** or **Fail** based on academic, demographic, and lifestyle factors. Using the UCI Student Performance dataset, the model analyzes various student attributes to identify patterns that influence academic success.

---

## 🎯 Objectives

- Predict student performance using machine learning techniques.
- Analyze factors affecting academic outcomes.
- Compare multiple classification algorithms.
- Evaluate model performance using standard metrics.
- Generate actionable insights from student data.

- Study time and previous grades significantly impact student performance.
- Higher absenteeism often correlates with lower academic achievement.
- Machine learning models can effectively identify students at risk of failing.
- Data-driven insights can help educators provide targeted support.
